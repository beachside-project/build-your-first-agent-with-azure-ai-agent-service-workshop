## はじめに

Bingによる会話のグラウンディングは、Azure AI Agent Service によって提供される複数のツールのひとつです。Bingによるグラウンディングを使用すると、アプリは、会話に関連し、かつベクトルストアや SQL DB でエージェントに事前に提供できない情報を検索できるようになります。  
例として、競合製品情報を検索したい場合などです。
